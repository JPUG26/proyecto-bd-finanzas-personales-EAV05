# Base de Datos - Aplicación de Gestión Financiera Personal

## Descripción del Proyecto
Este repositorio contiene el desarrollo completo de base de datos para una aplicación de finanzas personales. El proyecto está estructurado en tres entregables (Sprints) que cubren desde la definición inicial de reglas de negocio hasta la implementación de seguridad, triggers y procedimientos almacenados.

## 🚀 Entregables del Proyecto

### 📂 [Sprint 1 (10%): Modelado Inicial](./sprint1/)
*Estado: Completado*
1. **Definición de entidades** a partir de las reglas de negocio del módulo.
2. **Principales consultas** identificadas o preguntas a responder del módulo.
3. **Definición de modelo lógico** con entidades y relaciones definidas (MER).
4. **Modelo físico** con columnas y clave primaria (definido por el Product Owner).
